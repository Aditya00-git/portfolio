OVERVIEW
--------
This is my personal portfolio website built using pure HTML, CSS, and JavaScript.
The website showcases my projects, achievements, certifications, and includes
a guestbook powered by GitHub Discussions.

The design is inspired by clean developer portfolios and focuses on simplicity,
readability, and smooth user interactions.


FEATURES
--------
- Dark themed modern UI
- Animated white particles background
- Clean homepage with introduction and social links
- Projects section with GitHub and live demo links
- Certifications section
- Achievements section
- Contact section with animated email underline
- Guestbook page using GitHub Discussions (Giscus)
- Smooth hover effects and entrance animations
- Fully responsive (desktop and mobile)
- Deployed using GitHub Pages


TECHNOLOGIES USED
-----------------
- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome (icons)
- Giscus (GitHub Discussions for Guestbook)


PROJECT STRUCTURE
-----------------
index.html        -> Main portfolio page

style.css         -> Styling and animations

script.js         -> Particles, scroll effects, interactions

guestbook.html    -> Guestbook page

README.txt        -> Project information
